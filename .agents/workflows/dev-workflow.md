# New Project Initialization
1. Search and run `npx create-vite@latest ./ --template react` (non-interactive).
2. Setup `index.css` with a CSS Variable design system (Colors, Spacing, Typography).
3. Create a `layout/` directory with responsive `Header`, `Footer`, and `AppShell`.
4. Initialize a `manifest.json` for PWA readiness.

# Feature Implementation
1. Analyze requirements and create a design plan.
2. Build the UI component with internal state and styles.
3. Add business logic/services (API calls, data handling).
4. Verify mobile responsiveness and accessibility.
5. Update SEO metadata if the feature adds a new route.
